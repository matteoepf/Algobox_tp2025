# TP Culture Numérique – AlgoBox et GitHub

Ce projet a été réalisé dans le cadre du TP de Culture Numérique (2A).
Il a pour objectif de découvrir la gestion de versions avec GitHub
et le travail collaboratif autour d’algorithmes écrits en AlgoBox.

## Contenu du projet
- Dossier `src/` : fichiers source AlgoBox (.algo)
- Algorithmes simples utilisant variables, conditions et boucles

## Prérequis
- AlgoBox installé sur l’ordinateur
- Git (optionnel, pour cloner le dépôt)

## Exécution
1. Cloner le dépôt GitHub ou le télécharger en ZIP
2. Ouvrir AlgoBox
3. Ouvrir un fichier `.algo` depuis le dossier `src/`
4. Lancer l’exécution de l’algorithme
